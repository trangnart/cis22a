# 12.4 PRACTICE: Branches*: Shape display
![Easy] ![Difficult] ![Hard] ![Insane]

A user inputs the name of a shape (`square` or `triangle`).
The program outputs the shape using asterisks.

## Input
`square`

## Output
```
***
* *
***
```

## Input
`triangle`

## Output
```
  *
 * *
*****
```
